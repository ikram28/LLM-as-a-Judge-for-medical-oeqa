This repository will host the code, data, and evaluation pipeline used in our study on **LLM-as-a-Judge for French medical open-ended question answering (OEQA)**, including:

- Expert-annotated equivalence dataset  
- Evaluation scripts for LLM judges  
- Lightweight SFT and GRPO training code
